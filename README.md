# Plant Me, Pro!

## What is it?
This app was designed to allow gardeners to learn about best planting times and help keep track of when plants will sprout or be ready for harvest.

## Tech Stack Used

- React
- Express
- PostgreSQL
- Sequelize
- Flexbox
