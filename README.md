# <a href="https://frozen-sierra-13249.herokuapp.com/">Tennis Match</a>
For ILIS International

## What is it?

This app allows tennis players to match up with and record the results of games with other players.

## Tech Stack Used

- Express
- PostgreSQL
- Sequelize
- Passport
- Flexbox
